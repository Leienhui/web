# JavaScript进阶

## 一、函数

### 1、定义函数和调用函数

* 函数必须先定义，然后调用。

* ```javascript
  function hanshu(形参){
  
  }
  hanshu(实参);//调用函数
  ```

* 可以定义1次，调用多次

* 注意：函数如果遇见return语句了，就不会继续执行语句了。

### 2、函数可以有返回值

* ```javascript
  function sum(a, b) {
      return a + b;
  }
  var jieguo = sum(2, sum(3, sum(4, 5)));
  console.log(jieguo);
  ```

* 有返回值的优点：

  * 函数如果有返回值，那么函数调用的时候就可以当做值使用。

### 3、作用域

#### 1、局部变量

* JavaScript中的var ，叫做“函数级作用域”。

```javascript
	<script>
        function fun() {
            var a = 10;
        }
        fun();
        console.log(a); //报错
    </script>

```

* 变量在哪个函数中定义的，它就是这个函数中局部的变量。

#### 2、全局变量

* ```javascript
  var a = 10;
  function fun() {
      a = 20;   //全局变量a被重新赋值为20；只要函数一调用就赋值成功
  }
  
  fun();
  console.log(a);//20
  
  ```

* 全局变量没有定义在任何函数里面。

#### 3、遮蔽

```javascript
var a = 10;
function fun() {

   var a = 20;   //局部a把全局a遮蔽

  	a ++;

  	a ++;

  	a ++;

```

### 4、函数作用域

```javascript
function fun() {
    function sum(a, b) {
        return a + b;
    }
    console.log(sum(3, 4));
}

fun();//7
console.log(sum(3, 4));//报错:sum is not defined 

```

* 函数优先提升，整体提升； 变量随后提升，仅提升var。

### 7、数据类型

* 基本数据类型： null 、undefined 、String 、Number、Boolean

* 引用类型：object  function  数组  正则

* 基本数据类型比较的是值

* 引用数据类型比较的是地址

* 如果引用类型值`var arr2=arr1`不会产生副本，两个指向同一个地址，不克隆。

  ```javascript
  function type(n){
      n.b++;
  }
  var obj = {
      a: 1,
      b: 2,
      c: 3,
      k: 4
  }
  type(obj)
  console.log(obj.b); //3
  ```

  * **基本数据类型传值，引用类型传址**

* 进行**==**运算的时候，基本类型判**值**，引用类型判**址**

## 二、闭包

* **JavaScript规定：函数能够记忆住定义时候的作用域，即使函数不在定义的作用域中执行，则仿佛它的作用域也跟着它走了，它永远被定义时候的作用域影响。**

### 1、偏函数（高阶函数）

* 一个函数里面返回另一个函数就叫高阶函数

* ```javascript
  
          var a = 10;
          function outer() {
              var a = 20;
              // 返回匿名函数
              return function () {
                  alert(a);
              };
          }
          var inner = outer();//得到里面的函数
          inner();//执行里面的函数，20
    
  ```

* ```javascript
  
          var a = 10;
          function outer() {
              a = 20;
              return function () {
                  alert(a);
              }
              var a = 30;//提升声明，a便局部变量
          }
    
          var inn = outer();
          inn();//20
    
  ```

### 2、私有变量

* 

     ```javascript
     function fun() {
         var a = 10;
         return {
             // 下面的写法与  getA :function () {}写法一样
             getA() {
                 console.log(a);  //10
             }
         }
     }
     var obj = fun();
     obj.getA();
     ```

* 闭包返回一个对象，进行打点调用即可

* ```javascript
     function fun() {
         //这里的a是局部变量，要想再外面改变a的值只能通过调用add函数更改    
         var a = 10;
         return {
             // 下面的写法与  getA :function () {}写法一样
             getA() {
                 console.log(a);  //10
             },
             add() {
                 // 变量被私有化了
                 a++;
             }
         }
     }
     var obj = fun();
     obj.add();// 这先将a++
     obj.getA();  // 11
     ```
