# 基本类型

* number 数字， string字符串，boolean布尔，null,undefined是自己的类型
* undefined又是类型，又是值。这个类型只有一个值。

* **undefined**表示没有定义，但是不是报错，是一个特殊值。

* **定义的时候，如果不加var**，那么它就没有提升。

* ```
  console.log(c);   //p is not defined
  c=10;
  ```

* **事实上，如果一个变量只定义了，但是没有赋值，就是undefined**

* JS中不能定义一个“空变量”的。变量必须有值，undefined也是值。

### 2、类型转换

### 1、Number()转为数字

* **字符串转数字：**纯数字字符串能够转为数字，空字符串会被转为0；其他不纯字符串将转为NaN。

```javascript
console.log(Number('5'));//5
console.log(Number(''));//0
console.log(Number('5年'));  //NaN
console.log(Number(true));//1
console.log(Number(false));//0
console.log(Number(undefined));//NaN
```

* NaN是特殊的数字

### 2、Boolean()转为布尔

* **布尔值转数字：**true就是1，false就是0
* 【数字转布尔】，0和NaN就是false，非0就是true
* 【字符串转布尔】空字符串是false，非空字符串就是true
* 【undefined和null都是false】

```javascript
 console.log(Boolean(1));  //true
 console.log(Boolean(-34));  //true
 console.log(Boolean(0));   //false
 console.log(Boolean(NaN));  //false
```

### 3、String()转为字符串

* 所有东西用String()转为字符串，都是长成一个样子的字符串。

```javascript
(NaN).String();//"NaN"
```

### 4、隐式转换



* 做数学运算的时候，所有东西都会被Number()自动转。

* ```javascript
  console.log(3 + true);      	// 4
  console.log(3 + false);     	// 3
  console.log(3 * '8');     	// 24
  console.log(3 + '8');     	// '38'
  ```

* 有字符串，+就是连字符，就是拼接，不是加

* 做逻辑运算的时候，所有东西都会被Boolean()自动转。

### 5、说说NaN、undefined

#### 1、NaN



* NaN不是一个数，但是是数字类型的值。

* ## 0除以0等于NaN

* ```javascript
  console.log(0 / 0); //NaN
  console.log(340 / 0);//Infinity
  console.log(-10/ 0);//-Infinity
  
  ```

* ## NaN不自等

* ```javascript
  NaN==NaN  //false
  ```

* ## 如何检测是NaN？

* isNaN();

* ```javascript
  isNaN(NaN);//true
  ```

* 这个函数有bug，它的机理是，**所有**Number()**转之后，结果是NaN**的，都将被isNaN()函数判定为true。

* ## 什么时候得到NaN？？

* ```
  console.log('我' - '你');               // NaN
  console.log('我' * 3);                  // NaN
  console.log(parseInt('我爱你'));         // NaN
  ```

* ## NaN参与任何数学运算答案都是NaN

* **NaN与任何东西比较都是NaN**

#### 2、undefined

* #### undefined与上任何都等于undefined

* undefined或上任何都等于后面的

* **undefined转数字：**NaN

* **null转数字**为0

* undefined==null

* ```javascript
  console.log(undefined&&false)//undefined
  console.log(undefined&&true)//undefined
  console.log(undefined&&"hahahha")//undefined
  console.log(undefined&&"hahahah")//"hahahah"
  console.log(undefined||false)//false
  console.log(undefined||true)//true
  console.log(!undefined)//true
  ```

#### 3、parseInt()情况下的NaN

```javascript
parseInt('a123') // NaN 
parseInt('abc') // NaN 
parseInt('') // NaN 
parseInt(undefined) // NaN 
parseInt(null) // NaN 
parseInt(Infinity) // NaN

```
