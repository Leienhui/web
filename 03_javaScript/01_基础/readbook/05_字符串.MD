## 字符串

### 1、split()方法

* 一个字符串，可以调用split()分割为数组。

* ```javascript
  var str="abcd";
  var arr=str.split('');
  console.log(arr);//["a","b","c","d"];
  ```

  

### 2、substr()方法

* substr(开始下标，截取个数)；开始下标可以是负数

### 3、substring()方法

* substring(a下标，b下标)；

  * ，从a开始到b下标结束，但是不包括b这个位置,返回的子串会一直到字符串 的结尾

  * substring可以自动的交换两个参数的位置，保证小数在前面，大数在后面

    

### 4、slice()

* slice参数都是(a下标，b下标)，从a开始到b下标结束，但是不包括b这个位置,a下标可以是负数

### 5、charAt()

* charAt()方法可以得到指定位置的字符，位置序号从0开始

* 现在获取字符串的某一位可以直接使用[]

* ```javascript
  var str="abcdefgh";
  console.log(str[3])//d
  ```

### 6、 toUpperCase() 

* 将字符串转为大写 

### 7、toLowerCase() 

* 将字符串转为大写

### 8、 indexOf()方法

* 返回某个指定的字符串值在字符串中首次出现的位置 
* 如果要检索的字符串值没有出现，则该返回-1
* indexOf()方法还可以接收第二个参数，表示从第几位开始寻找

```javascript
'abcdeb'.indexOf('b') // 1 
'abcdeb'.indexOf('de') // 3 
'abcdeb'.indexOf('m')b // -1
'abcdeb'.indexOf('b'，2) // 5

```
