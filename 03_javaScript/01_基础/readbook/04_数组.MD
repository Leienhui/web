# 数组

## 数组的定义

* **数组**（array）就是一组数，用[]包起来，逗号隔开。

* 数组里面的项时字符串时，字符串一定要用引号包起来。

* ```
  <script>
      var arr = ['白板', '幺鸡', '二万', '三饼', '四万'];
  </script>
  ```

## 下标

* 我们用方括号里面加一个整数，表示数组的**下标**。这叫做**“枚举”**。

* ```javascript
  var arr = ['白板', '幺鸡', '二万', '三饼', '四万'];
  console.log(arr[0]);
  console.log(arr[1]);
  
  ```

* **下标从0**开始。一定要记住。

* 数组的长度用arr.length表示。**由于从0**开始数，所以最后一项下标一定是**arr.length - 1**

## 数组的头尾操作

* push()    尾部增加
* pop()    尾部删除
* unshift()  头部增加
* shift()   头部删除

* arr打点调用push，我们管push叫做数组的“方法”

## includes方法

* includes方法用来判断项在不在数组中。
* 求两个数组的交集

```javascript
<script>
        var arr1 = [1, 2, 3, 45,'hahah' , 56, 67, 8, 9, 100, 11];
        var arr2 = [12, 2, 32, 45, 'hahah', 561, 67, 81, 9, 100, 111,9000];
        var short = 0;
        var same = [];
        if (arr1.length > arr2.length) {
            short = arr2.length;
        } else {
            short = arr1.length;
        }
        for (var i = 0; i < short; i++) {
            if (arr1.includes(arr2[i])) {
                same.push(arr2[i]);
            }
        }
        console.log(same);
    </script>
```

## splice方法

* splice(开始项的索引，删除多少项，替换字符)；
* splice(1, 2, '★');  从下标为1的项开始，删除2项，**替换**为★
* splice(1, 0, '★');  将★**插入**到数组下标为1的位置
* splice(1, 2);     **删除**下标为1的项开始，连续2项都删除
* **注意：**splice会把它删除的东西返回

## indexOf()方法

* 检查数组中是否有某项，如果有，则返回最早出现的位置的下标，如果没有，返回-1。

## reverse()方法

* 可以让数组逆序。

## join()方法

* 可以让数组合并为字符串。

* 数组如果join()里面是空白，则会默认用逗号分割，组建字符串。
* 如果是join('')则没有空隙，组建字符串。

## slice()子数组

* slice(开始下标, 结束下标)表示从原来的数组中，截取下标为**开始下标**的项到**结束下标**，但不包含**结束**的项。

## concat() 方法

* 用于连接两个或多个数组。该方法不会改变现有的数组，而仅仅会返回被连接数组的一个副本。

  ```javascript
  <script type="text/javascript">
  var a = [1,2,3];
  document.write(a.concat(4,5));
  console.log(a);  //[1,2,3],但是页面上显示的是1，2，3，4，5
  </script>
  ```

* ```javascript
  var arr1 = [1];
  var arr2 = [3, 4];
  var arr3 = [9];
  var arr4 = arr1.concat(arr2, arr3, arr2, arr1);
  console.log(arr4);  //[1, 3, 4, 9, 3, 4, 1]
  console.log(arr1);  // 不变，还是[1,2,3]
  ```

## sort()

           return zero(a)-zero(b);

```
