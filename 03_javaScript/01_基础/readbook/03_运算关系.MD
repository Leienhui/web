# 运算关系

## 取余

* JS中用%表示取余数。

* 注意，百分号运算符，不关心整除部分是几，它只关心余数

* ```javascript
  10 % 3
  ```

* Math.ceil(数字)，将返回这个数字在数轴上右边第一个整数，即大于它的第一个整数。当然如果它自己就是整数，则返回它自己。

## parseInt()内置函数

* 这个函数可以把小数变为整数，怎么变，截断小数点后的数字。它不四舍五入。

* 也就是说，对于正数来说Math.floor()的结果一定和parseInt()相同。但是，对于负数来说，负的整数Math.floor()的结果一定和parseInt()相同，但是负的小数Math.floor()要比parseInt()的结果小1。

* 事实上parseInt()函数的功能非常丰富。如果给parseInt()一个字符串，那么它能够把字符串变为数字，并且能够将字符串尾巴的汉字（非数字字符）去掉。

* 不是数字的东西必须在尾巴上。否则出现NaN。

  * ```javascript
    var b=parseInt('你是1111');//NaN
    var a=parseInt('1111哈哈哈哈哈哈');//1111
    ```

* 对于一个二进制的数字的字符串来说，parseInt()函数可以将它转为10进制的对应数字。

  * ```javascript
    var b=parseInt('1101',2);//13
    ```

## 关系运算

* \> 大于号

* < 小于号

* == 等于
* \>= 表示大于或等于

* <= 表示小于或等于
* \>= 表示大于或等于

* <= 表示小于或等于

* 进行关系运算的结果一定是true或者false。
* 注意==表示判断是否相等，而一个等号=表示赋值。三个等判断是否全等，就是说，数值是相同的，类型也是相同的。
* 进行等等判断的时候，如果两个值“隐式转换”（内部的类型转换）结果相同，也是true。 比如0和false是等等的，但是不是等等等的。

## 逻辑运算

* `! `表示非，也叫作“置反运算”，它可以把true变为false，把false变为true。

* `&&` 表示且，“都真才真”

* `||` 表示或，“有真就真”

* `^`在js中表示按位异或，是进制的运算符，相同为0，不同为1

* ```javascript
  7^5  //结果为2
  ```


## 短路计算

* **注意：**要左右两边有bool值结果才是布尔值，否则就是短路

  ```javascript
  4||16  //结果是4
  3&&8 //结果是8
  ```

* a && b的时候，

  - 如果a是false，那么无论b是什么，总结果一定是a。
  - 如果a是true，那么b就是总结果。a是true，b就万众瞩目了，b真总结果就真；b假总结果就假。

* 进行逻辑运算的时候，结果不一定是布尔值

* 不是0的数字就是真

* a || b的时候，

  - 如果a是false，那么b就是总结果。a是false，b就万众瞩目了，b真总结果就真；b假总结果就假
  - 如果a是true，那么总结果一定是true，a就是总结果。

## 三元运算符

* ```javascript
  function fib(n){
      return n==0||n==1?1:fib(n-1)+fib(n-2);
  }
  console.log(fib(9));
  ```

  ```javascript
  表达式?表达式为true的值:表达式为false的值
  ```

* 总顺序： 数学、关系、逻辑!

* 逻辑里面，先非（！）、然后与（&&）、最后（||）。


## 引用类型值和基本类型值（重要）

* 区别有两个：

  * ① 变量传值的时候，基本类型值传副本；引用类型值，传地址；

  * ② ==判断的时候，基本类型值看值是否等；引用类型值，看是否是同一个地址。

  * ```javascript
    var a = [33, 44, 55];
            var b = a;
            a.push(66);
            console.log(b);
    
    ```

* 基本数据类型值，如果进行变量传值，则会复制一份。

* 引用类型值，如果进行变量传值，会指向同一个堆内存，而不是复制一份。

* ==判断的时候，基本类型值看值是否等；引用类型值，看是否是同一个地址。

* ```javascript
  console.log(3 == 3);                        // true
  console.log([1, 2, 3] == [1, 2, 3]);       // false
  var arr = [];
  console.log(arr == []);       // false
  console.log(arr.length == 0);       // true
  ```

## ++运算符

* a++ 先用再自增

* ++a 先自增再用

* ```javascript
  var a = 5;
  console.log(a++);   //5
  console.log(a);     //6
  ```

* ```javascript
  var a = 5;
  console.log(++a);       //6
  console.log(a);         //6
  ```
