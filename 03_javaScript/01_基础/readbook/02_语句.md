## 多分支if

* ```javascript
  // 给按钮添加监听
  btn.onclick = function () {
      // 得到用户分数
      var score = Number(kuang.value);
  
      if (score >= 85) {
          alert('优秀');
      } else if (score >= 75) {
          alert('良好');
      } else if (score >= 60) {
          alert('及格');
      } else {
          alert('不及格');
      }
  };
  
  ```

* 最后有一个else分支，必须放到最后。当然也可以没有。

* 为什么可以不写&&。因为“跳楼现象”。

  * 就是一个小朋友站到站到楼顶。如果**满足了其中的一个分支条件，就会立即跳楼，不会再执行后边的任何条件**。也就是说如果他走到了这个楼层，那么就是说它不满足之前的所有分支条件。

    var nan = document.getElementById('nan');

## for循环

* ```javascript
  for(var i=0;i<5;i++){
  	for(var j=0;j<5;j++){
  	
  	}
  }
  ```

* 首先判断第一层循环，0<5为true，执行内层循环，内层循环执行完毕以后，i++，判断1<5为true,执行内层循环，内层循环执行完毕以后，i++，判断2<5为true......

## while循环

* 语法：

  * ```
    while(表达式){
    语句
    }
    
    ```

  * while是“当”的意思，当表达式成立，就会做语句。做完语句再判断一次表达式是否成立，当表达式成立，就会做语句。做完语句再判断一次表达式是否成立，当表达式成立，就会做语句。做完语句再判断一次表达式是否成立，当表达式成立，就会做语句……,遇到`break;`或者表达式不成立，才能跳出循环

* ## 什么时候用while，什么时候用for？

  * 答案特别简单：知道循环范围、次数的，用for； 不知道循环次数的用while。

## switch()语句

```javascript
switch(表达式){
    case 子句1:
    语句体1
    break;
    case 子句2:
    语句体2
    break;
    case 子句3:
    语句体3
    break;
    case 子句4:
    语句体4
    break;
    case 子句5:
    语句体5
    break;
}
```

* 如果某个子句和表达式时相同的，那么会一直执行下面的语句体，**不管其他子句满不满足条件**，直到遇见一个break；
