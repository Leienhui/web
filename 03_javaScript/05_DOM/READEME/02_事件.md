# 事件（难点）

* 人为规定，事件是：
  * 从最外层往最内层传（捕获阶段）
  * 从最内层往最外层传（冒泡阶段）



## 事件冒泡与捕获

```html
 <div class="box1">
        <div class="box2">
            <div class="box3">
                <button class="btn">按钮</button>
            </div>
        </div>
    </div>
```

* 从box1到btn的过程**是捕获**
  * box1--->box2--->box3--->btn
* 从btn到box1的过程**是冒泡**
  * btn--->box3--->box2--->box1
* **注意：**最内层不区分冒泡捕获，谁写在上面先运行谁
  * 例如：在编写js的时候，最内层先编写捕获代码，那么最内层就是先捕获；如果最内层先编写冒泡代码，那么最内层就是先冒泡
* 大多数**on事件**都是**监听冒泡阶段**，除onmouseenter/onmouseleave天生不继续传播
  * 这种叫DOM0级

```javascript
<script>
        var btn=document.getElementById('btn');
        var box1=document.getElementById('box1');
        var box2=document.getElementById('box2');
        var box3=document.getElementById('box3');
        btn.addEventListener('click',function(){
            alert("我是btn");
        },true)//写true表示捕获阶段
        box1.addEventListener('click',function(){
            alert("我是box1");
        },true)//写true表示捕获阶段
        box2.addEventListener('click',function(){
            alert("我是box2");
        },true)//写true表示捕获阶段
        box3.addEventListener('click',function(){
            alert("我是box3");
        },true)//写true表示捕获阶段
</script>
```

* 下面的写法表示DOM2级监听写法

* ```javascript
  btn.addEventListener('事件',function(){},true)//第三个参数为true表示监听捕获阶段
  btn.addEventListener('事件',function(){},false)//第三个参数为false表示监听冒泡阶段
  ```

  * 参数1：表示事件
  * 参数二是一个函数，表示要做的事情
  * 参数三是一个Boolean值；如果不指定第三个参数，默认为false

## onclick与addEventListener冒泡阶段的区别

```javascript
 btn.onclick=function(){
            console.log("A");
        }
btn.onclick=function(){
            console.log("B");
        }
```

* 上面的代码只会输出B
* onclick后写的会覆盖先写的

```javascript
btn.addEventListener('click',function(){
            console.log("A");
        },false)
btn.addEventListener('click',function(){
            console.log("B");
        },false)
```

* 上面的代码会输出A,B
* addEventListener不会覆盖

## IIFE（立即执行匿名函数）（重要）

* 一个函数没有名字，被一个圆括号包裹，然后又再被一个原阔号调用就叫IIFE

* ```javascript
  (function (){
  
  })();
  ```

* 包裹函数的圆括号表示将函数变为表达式

* 最外面的圆括号表示调用函数

* ```javascript
  (function (a){//这里的a是形参
  
  })(m);//这里的m是实参
  ```

* 最外面的圆括号可以有参数，参数可以在函数中调用

* ```javascript
  for(var i=0;i<10;i++){
      btn[i].onclick=function(){
      console.log(i);//无论点击哪个按钮，输出的i始终是9
      }
  }
  ```

* ```javascript
  for(var i=0;i<10;i++){//当循环语句用var 来声明的时候这个函数就变成了闭包
      btn[i].onclick=(function(i){
      console.log(i);//点击的按钮是几，输出的结果就是几
      })(i);
  }
  ```

## 事件委托（面试必考）

* **事件委托：**就是利用**事件冒泡**，将元素的事件委托给**祖先元素**，**一般是父亲**

* 什么时候使用事件委托

  * ①遇到给大量类似的元素添加监听，内存消耗大

  * ②动态上树的新元素,没有事件监听

* ```javascript
  box.onclick=function(e){
      if(e.target.nodeName.toLowerCase()=='li'){//如果标签名为li，那么让其背景变红
          //e.target表示点击的最内层元素
  		e.target.style.background='red';
      }
  }
  ```

  

## 阻止事件传播

* **e.stopPropagation()**阻止事件继续传播

* 问题：点击按钮弹出抽屉，点击页面任何一个地方关闭抽屉

* 方法1：阻止事件冒泡

  ```javascript
   // 由于在冒泡事件中，btn也是属于document的，所以如果不阻止事件继续传播，那么点击按钮的时候执行的函数还是下面那个
  btn.onclick=function(e){
              // 阻止事件继续传播，事件也不会往document上传播了
              e.stopPropagation();
              box.style.right=0;
          }
  document.onclick=function(){
              box.style.right=-300+"px";
          }
  	//点击box会阻止事件往document上传播
  box.onclick=function(e){
              e.stopPropagation();
          }
  ```

* 方法2：采用事件捕获

  ```javascript
   btn.addEventListener('click',function (){
              box.style.right=0;
          },true);
  document.addEventListener('click',function (){
              box.style.right=-300+"px";
          },true)
  box.addEventListener('click',function (){
              box.style.right=0+"px";
          },true)
  ```

## 特殊事件

* onmouseenter/onmouseleave天生不继续传播
* onmouseover/onmouseout会继续传播
* **onmouseenter天生不冒泡，onmouseover冒泡，要阻止冒泡用e.stopPropagation()**
