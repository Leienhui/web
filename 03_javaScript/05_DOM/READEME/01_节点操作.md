# DOM

## 得到节点的方法有5种

* document.getElementById()
  * 通过id得到节点
* document.getElementsByTagName()
  * 通过标签名得到节点，是一个数组
* document.getElementsByClassName()
  * 通过类名得到节点，是一个数组
* 兼容性查询网站：caniuse.com

###   通过选择器获取节点

* document.querySelector('#p1');
  * 获取id为p1的元素
* document.querySelectorAll('p');
  * 获取所有为p的标签，是一个数组

## 节点与节点之间的关系

* 父节点：parentNode

* 子节点：childNodes

  * 得到的是数组

* 第一个子节点：firstChild

  * (会将空格、换行算作节点，会出现undefined错误),**如果要使用该方法，就需要通过nodeType判断，如果nodeType==1，说明是普通节点，而不是文本节点，如果nodeType==3那么就是空文本节点，下面的其他方法亦是如此**
  * 所以要获取第一个孩子用：
    * firstElementChild

* 最后一个子节点：lastChild

  * (会将空格、换行算作节点，会出现undefined错误)
  * 所以要获取第一个孩子用：
    * lastElementChild

* 前一个兄弟：previousSibling

  * (会将空格、换行算作节点，会出现undefined错误)

  * 所以要获取前一个兄弟用：
    * previousElementSibling

* 后一个兄弟:nextSibling(会将空格、换行算作节点，会出现undefined错误)

  * 所以要获取后一个兄弟用：
    * nextElementSibling

### 修改自定义属性

#### 获得属性值--getAttribute("自定义属性名")

```javascript
<button id="btn"></button>
    <p data="17"></p>
    <script>
       var btn=document.getElementById("btn");
       var p=document.getElementsByTagName("p");
       btn.onclick=function(){
        var a=p[0].getAttribute('data');//得到自定义属性值
        console.log(a);

       }
```

#### 设置属性值--setAttribute('属性名'，属性值)

```javascript
   <button id="btn"></button>
    <p data="17"></p>

    <script>
        var btn = document.getElementById("btn");
        var p = document.getElementsByTagName("p");
        var a = parseInt(p[0].getAttribute('data'));
        btn.onclick = function () {
            var b = p[0].setAttribute('data',++a);
        }
    </script>
```

## 创建节点

```html
 <input type="text" value="哈哈哈哈" id="input">
 <button id="btn">按钮</button>
 <ul id="ul"></ul>
```

```javascript

    <script>
    //获得节点
        var input=document.getElementById('input');
        var btn=document.getElementById('btn');
        var ul=document.getElementById('ul')
        btn.onclick=function(){
        //创建节点
            var li=document.createElement('li');
            //往创建的节点之中添加内容
            li.innerText=input.value;
            //将创建的孤儿节点添加父节点
            ul.appendChild(li);
        }
    </script>
```

## 父元素.insertBefore(新元素，标杆元素)

```javascript
var p=document.createElement('p');
p.innerText='呀呀呀呀';
box.insertBefore(p,ul);//在box里的ul前面插入一个p标签
```

## 删除元素

* 要自杀，找父亲

  * 只有父亲能删除子元素

  * ```javascript
    父元素.removeChild(子元素);
    box.removeChild(ul);//删除box里面的ul
    ```

  * ```javascript
    ul.parentNode.removeChild(ul);//建议用这个删除
    ```

* 如果父亲是body/head的时候，要删除里面的子元素：

  * ```javascript
    document.body.removeChild(子元素)
    document.head.removeChild(子元素)
    ```

##  移动元素

* appendChild()和insertBefore()
* 一个节点绝对不能同时有两个父亲，所以要换父亲

```javascript
 box2.appendChild(p1);   //移动为box2的最后一个子元素
 box2.insertBefore(p1,p3);  //移动到box2里面p3元素之前
```

## 克隆节点

* `元素.cloneNode(true)`参数设为true表示深克隆，不加是浅克隆

* ```javascript
  // 克隆出来的节点都是孤儿节点，
  var p = p1.cloneNode(true);
  // 孤儿节点上树，注意：原盒子中的p1还存在。只是多多复制了一份
  box2.appendChild(p);
  ```
