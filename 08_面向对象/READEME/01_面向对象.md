# 面向对象

## 函数上下文（this）

* this不表示函数本身，this是函数的**上下文**。要根据函数调用时候的**前言后语**决定

### 规则1：对象打点调用函数，函数的this是这个对象

```javascript
function fun() {
    console.log(this.a);	//输出为399，这里的this指的是obj
}

var obj = {
    a: 399,
    fn: fun
}

obj.fn(); //在这里对象打点调用函数

```

* 把函数弄到对象里面了，目的是让函数可以以“对象.函数()”的形式调用，此时函数有了this上下文，就是那个对象。





* 函数有没有上下文不是看他去过哪儿，而是看“临门一脚”：看射门那一下，谁是射门那人。

```javascript
function fun() {
    console.log(this.a);	//undefined
}
var obj = {
    a: 399,
    fn: fun
}

fun();

```











```javascript
var obj = {
    a: 3434,
    fn: function(){
        alert(this.a);  ////undefined,这里的this找不到上下文，也就是找不到射门人
    }
}

var fn = obj.fn;
fn();

```





```javascript
var obj = {
    a: 3434,
    fn: function(){
        alert(this.a);	//结果为3434，this就是这个对象
    }
}

obj.fn();		//	对象直接打点调用函数
```





```javascript
var obj = {
    a: 3434,
    fn: function(){
        alert(this.a);  //	没调用函数，不知道this是谁，也就是射球的人是谁。
    }
}
```

### 类数组对象

* **本质上是个对象，但是属性名都是0、1、2、3、4...，并且有length属性**。

```javascript
var obj = {
    0: '小明',
    1: '小红',
    2: '小强',
    3: '小花',
    length: 4
};

console.log(obj[0]);	//	小明
console.log(obj.length);	//	4

```

* **一定记住，类数组对象，不能.0**,必须使用**obj[数字]**;

* 因为0不是标准的键名，所以只能[0]。JS给他们自然数开了一个特例。此时，特别像数组。但是不能用数组的方法，push()都不能用！！

* arguments： 函数的实参清单。

  ```javascript
  function fun() {
      console.log(arguments);		
  }
  
  fun(22, 33, 44, 55)
  
  ```

  ```javascript
  //上面函数中的输出结果是
  {
  0: 22,
  1: 33,
  2: 44,
  3: 55,
  length: 4
  }
  
  ```

  * 调用函数的时候，传入了4个参数，这4个参数就汇总到了一个类数组对象里面。

* ```javascript
  function fun1(a, b, c) {
      arguments[0](); //函数的this是这个类数组对象
  }
  
  function fun2(a, b, c, d, e) {
      alert(this.length);	// 7，这里的this就是fun1的arguments
  }
  
  
  fun1(fun2, 66, 77, 88, 99, 44, 33);
  
  ```

  * **arguments.callee表示函数本身，而函数的长度是形参列表长度**

  * 函数.length是形参列表长度

* 

  ```javascript
  function fun1(a, b, c) {
      arguments[0](8,9);
  }
  
  function fun2(a, b, c, d, e) {
      alert(this.length);                 //7
      alert(arguments.length);            //2
      alert(arguments.callee.length);     //5
      alert(this.callee.length);          //3
  }
  
  
  fun1(fun2, 66, 77, 88, 99, 44, 33);
  
  ```

```javascript

```javascript

## 用new调用函数

* **用new可以调用函数，此时函数的上下文this特别奇怪，是一个秘密创建的空白对象{}。并且，函数会自动返回这个this对象。**

### 四步走：

① 秘密创建空白对象

② this指向这个对象

③ 执行函数

④ 返回this

```javascript
function fun(){
    this.a = 10;
    this.b = 20;
    this.c = 30;
}
var obj = new fun();
console.log(obj);	//	{a:10,b:20,c:30}
```

* 因为new函数了，所以立即执行“四步走”：

  ① 首先秘密创建空白对象，{}

  ② 函数的this指向创建空白对象

  ③ 执行语句，所以空白对象不是空白了，就是{a:10,b:20,c:30}了。

  ④ 返回this

## 构造函数



```javascript
        // 构造函数 
        function People(name, age, sex){
            this.name = name;
            this.age = age;
            this.sex = sex;
        }

        var xiaoming = new People('小明', 12, '男');
        var xiaohong = new People('小红', 11, '女');
        var xiaogang = new People('小刚', 13, '男');

        console.log(xiaoming);	//	{name:"小明",ge:12,sex:"男"}
        console.log(xiaohong);	//	同理
        console.log(xiaogang);	//	同理
```

* 用new调用函数，会四步走。所以每次都会返回一个带有name、age、sex的对象。

  * 这些对象都有同样的属性名，所以叫做“同一类对象”。

  * People函数，就是“**构造函数**”。非常像Java中的**“类”**。xiaoming、xiaohong、xiaoqiang就好比是Java中的**“实例”**。

  * **构造函数首字母通常大写**。

### 类与实例

* 类：**描述了这一类的东西应该有什么**。

```javascript
function People(name, age, sex) {
    this.name = name;
    this.age = age;
    this.sex = sex;
}
```

* 上面的函数虽然没有具体的值，但是描述了应该有什么值。



* 实例是真实的东西：

```javascript
        var xiaoming = new People('小明', 12, '男');
        var xiaohong = new People('小红', 11, '女');
        var xiaogang = new People('小刚', 13, '男');
```

* 所以=>人是**类**，小明、小红、小刚是**实例**

### 原型prototype

* 任何函数都有原型：

```javascript
function sum(a, b) {
    return a + b;
}

console.log(sum.prototype)	//	{constructor: ƒ}
```

* prototype就是函数的原型
* 函数都有原型，有用么？？没用！！ 只有当函数是构造函数的时候，才有用。

* **构造函数的原型，就是new出来的实例的原型链**
* **原型链具有原型链查找的能力**

```javascript
        function People(name,age,sex){
            this.name = name;
            this.age = age;
            this.sex = sex;
        }
        People.prototype.singing = function(){
            console.log('我' + this.name + '，今年' + this.age + '岁啦');	//	我小明，今年12岁啦
        }
        People.prototype.eat = function(food){
            console.log('我是' + this.name + '，我在吃' + food);	//	我是小明，我在吃葡萄	
        }
        People.prototype.sleep = function(food){
            console.log('我是' + this.name + '我睡觉了');
        }
       
        var xiaoming = new People('小明', 12, '男');

        xiaoming.singing();
        xiaoming.eat('葡萄');
```

* **构造函数的prototype就是实例的`__proto__`。**

  **而`__proto__`叫做原型链，可以进行原型链查找。**
