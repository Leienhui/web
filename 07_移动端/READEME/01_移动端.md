# 移动端

## 视口技术

* 让手机觉得自己没有那么宽

  `<meta name="viewport" content="width=device-width, initial-scale=1.0">`

  * `viewport`:表示视口
  * `width=device-width`:宽度等于设备的宽度

* **背下来**：

* ```html
      <meta name="viewport" content="width=device-width">
  ```

* ```javascript
  // 打印出屏幕宽度
  $('#zzz').html(window.width());
  ```

* 

## 屏幕的宽高

```javascript
var h1 = document.getElementById('h1');
// document表示文档
// documentElement表示文档根元素，也就是屏幕
// clientWidth表示宽度
// clientHeight表示高度
h1.innerHTML = document.documentElement.clientWidth;
h1.innerHTML = document.documentElement.clientHeight;
```

### box-sizing(面试必考)

* 表示盒子模型尺寸设置
* `box-sizing:border-box;`
  * 让盒子的宽度涵盖border边框
  * 即**内容盒、padding、border都在width里面了**
* 所以以后**做缝隙**要用边框来模拟

### 固比固的制作

```css
 <div class="box6">
        <div class="grid1"></div>  → 绝对定位 width:60px;
        <div class="grid2"></div>  → width:100%;
        <div class="grid3"></div>  → 绝对定位 width:60px;
 </div>

```

## flex布局（重要）

### 基本使用

```css
<div class="box1">      → display:flex; 儿子自动浮动了 	弹性盒容器

    <div class="grid">1</div>   → flex:1;
    <div class="grid">2</div>   → flex:1;
    <div class="grid">3</div>   → flex:1;
    <div class="grid">4</div>   → flex:1;
</div>
<div class="box2">  	→ display:flex; 儿子自动浮动了 弹性盒容器
    <div class="grid no1">1</div>       → flex:2;
    <div class="grid no2">2</div>       → flex:2;
    <div class="grid no3">3</div>       → flex:1;
    <div class="grid no4">4</div>       → flex:1;
</div>
```

* `display:flex;` 一定是给父亲用的，它自己就是弹性盒容器了，里面内容自动浮动了。
* flex:1 表示分配比例，给儿子用的

### 自动涵盖border

```css
<div class="box3">  	→ display:flex; 儿子自动浮动了 弹性盒容器
    <div class="grid no1">1</div>       → flex:1; 放心大胆写border:1px solid red;
    <div class="grid no2">2</div>       → flex:1; 放心大胆写border:1px solid red;
    <div class="grid no3">3</div>       → flex:1; 放心大胆写border:1px solid red;
    <div class="grid no4">4</div>       → flex:1; 放心大胆写border:1px solid red;
</div>

```

### 自动算margin

```css
<div class="box4">  → display:flex; 儿子自动浮动了 
    <div class="grid">1</div> → flex:1; margin-right:10px;
    <div class="grid">2</div> → flex:1; margin-right:10px;
    <div class="grid">3</div> → flex:1; margin-right:10px;
    <div class="grid">4</div> → flex:1; margin-right:0px;

</div>

```

### justify-content

* | 值            | 描述                         |
  | :------------ | :--------------------------- |
  | flex-start    | 默认值。从左到右开始排列     |
  | flex-end      | 从右到左开始排列             |
  | center        | 居中排列。                   |
  | space-between | 中间空格距离相等，两边无空格 |
  | space-around  | 两边与中间空格距离相等       |

### flex-direction

* | 值             | 描述                                              |
  | :------------- | :------------------------------------------------ |
  | row            | 默认值。从左到右水平显示  例如：水平方向上 ：1234 |
  | row-reverse    | 从右到左水平显示    例如：水平方向上 ：4321       |
  | column         | 从上到下竖直显示 例如：竖直方向上 ：1234          |
  | column-reverse | 从下到上竖直   例如：竖直方向上：4321             |

## rem布局

* 用于做宽度高度等比变化

* rem就是**html标签字号的倍数**

  ```css
  html{
  font-size:40px;
  }
  .box{
  width:3rem;		//那么盒子的宽度就是40*3=120px
  height:3rem;	//那么盒子的高度就是40*3=120px
  }
  ```

* rem的缺点：这一行都必须以rem为单位，不能用px为单位

## 媒体查询

* 看博客：https://blog.csdn.net/SunShinessx/article/details/89338008

* @media 可以针对不同的屏幕尺寸设置不同的样式
* 媒体查询: 通过查询当前属于哪种设备, 让网页能够在不同的设备下正常的预览

```css

html {
    font-size: 32px; /* 320/10 */
}
body {
    font-size: 16px; /* 修正字体大小 */
    /* 防止页面过宽 */
    margin: auto;
    padding: 0;
    width: 10rem;
    /* 防止页面过宽 */
    outline: 1px dashed green;
}
 
/* js被禁止的回退方案 */
@media screen and (min-width: 320px) {
    html {font-size: 32px}
    body {font-size: 16px;}
}
@media screen and (min-width: 481px) and (max-width:640px) {
    html {font-size: 48px}
    body {font-size: 18px;}
}
@media screen and (min-width: 641px) {
    html {font-size: 64px}
    body {font-size: 20px;}
}

```

* 可以再媒体查询的时候设置不同屏幕的html字号的大小，下面通过rem来进行调用

* 小知识：使用rem的时候，页面上不能出现别的杂质，因此使用雅虎的清除所有样式

  ```css
  <link rel="stylesheet" 
  type="text/css" href="http://yui.yahooapis.com/3.18.1/build/cssreset/cssreset-min.css">
  ```

* 手机中的图片必须至少是使用尺寸的两倍

  * 例如图片的真实宽高是40x30，再手机网页中我们使用的是20x15

  * 如果是精灵图，那什么都要除以2，例如：background-size,background-position

    * 例如：实际的精灵图是400x400;所以在设置精灵背景尺寸的时候，只需要设为200x200
    * `background-size:200px 200px`
