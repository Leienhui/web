# markDown语法

##   1、使用`=`和`-`标记一级和二级标题。

* ```
  一级标题
  =========
  二级标题
  ---------
  ```

## 2、使用`#`，可表示1-6级标题

## 3、单行代码\`代码`

## 4、多行代码\```

## 5、嵌套`>`

```
> 第一层
>> 第二层
>>> 第三层
```

## 6、强调

```
*斜体*，
**粗体**，
***斜体加粗*** ，
~~删除~~
```

## 7、图片

```
![img](url){:class=“responsive”}
![img](url){:height=“50%” width=“50%”}
![img](url){:height=“100px” width=“100px”}
```
