# CSS3

##  过渡

* **transition**
* ` transition: all 3s ease 0s;`
  * all：表示所有属性参与过渡
  * 3s：表示动画时长。动画时长的单位必须是s
  * ease:  表示两边慢中间快，是一个动画缓动策略,linear表示匀速
  * 第四个参数：表示延迟时间，如果不延迟，写0s，不能写0
  * 还可以进行背景颜色的过渡

* 过渡的效率高于定时器

## 2D变形属性

* **transform**
* `      transform: rotate(30deg);`
* 正方向写正的，负方向写负的

### 斜切变形

* `      transform:skew(10deg,20deg);`
* `transform:skew(横向度数，纵向度数)；`

###       缩放变形

* `transform: scale(2);`
  * 将盒子放大两倍

## 3D变形

* 任何3D变形都必须有一个”舞台“

* 舞台必须设置perspective属性，表示“观测距离“

* **如果你要旋转整个3d结构，那么舞台必须加：**

  **transform-style: preserve-3d;**

*  `transform: rotateX(30deg); `
  
  * 正方向是后仰
  
*  `transform: rotateY(30deg);`
  
  * 右肩向前扭动
  
* translateX()表示往右，正数往右，负数往左

### 空间移动

* translate();
* 让盒子沿着当前的z轴前进
* `      transform: rotateX(30deg) rotateY(30deg) translateZ(100px);`
* 注意：`translateZ(100px)`
* 旋转的时候z轴方向总是垂直与盒子;translateZ很关键，表示向自己当前朝向移动

## 保留3D

* 如果一个盒子内部有不同旋转的元素，自己也要旋转的情况下，这个盒子必须设置保留3D的属性，即“自己即是容器又是演员”
* `transform-style:preserve-3d;`

## 定义动画

* ```css
  @-webkit-keyframes dong{
      from {
          left:0;
      }
      to {
          left: 800px;
      }
  }
  
  ```

* @表示定义

* ## 调用

### 基本使用：

`animation: dong 2s ease 0s;`

* 参数意义和transition相同。

 

* 指定动画次数：

`animation: dong 2s ease 0s 3;`

* 执行3次

 

* 无限次：

`animation: dong 1s ease 0s infinite;`

* 执行无限次

 

* 偶数次逆向返回：

`animation: dong 1s ease 0s infinite alternate;`

 

* 最后停在原地：

`animation: dong 1s ease 0s forwards;`

## 落地页的制作

### zepto库

* zepto库就是将jQuery和过渡进行了整合：

① 用的jQuery的语法

② 但是机理全面变成了过渡。

* jQuery的animate不能进行颜色改变的动画。

### zepto库使用

* zepto简单好学，语法和jQuery一模一样。
* zepto底层不是定时器，机理：把普通语法变为过渡属性。
* zepto因为底层比较牛逼，所以能够做border-radius动画、背景颜色改变动画。
* zepto不支持eq，但是支持:nth-child。

* **注意：**

  * :nth-child(2n)

  * 表示偶数

   

  * :nth-child(2n + 1)

  * 表示奇数

```javascript
	<script src="http://www.zeptojs.cn/zepto.min.js"></script>

	<script type="text/javascript">
		// 当前你已经到过的最大的楼层编号，从1开始。
		var maxfloor = 1;

		// 监听窗口的滚动事件，这里有一个坑：window不能加引号

		$(window).scroll(function () {
			// 加入没有到底
			if (maxfloor < 5) {
				// 当前的已经卷动的高度
				var nowScrollTop = $(window).scrollTop();
				// 下一楼层的净高度
				var nextFloorTop = $('.bigbox.no' + (maxfloor + 1)).offset().top;
				// 当前在1层，关心是否到2层了；当前在2层就关心是否到3层。
				if (nowScrollTop > nextFloorTop) {
					// 成功来到下一层
					maxfloor++;
					console.log('欢迎光临' + maxfloor);
				}
			}
		});
	</script>

```
