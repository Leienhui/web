## 二、CSS常用选择器

###  1、标签选择器

* 标签选择器就是标签的名字，比如p、h1就是标签，它们表示

  选择页面上**所有**这个标签。

* 标签选择器杀伤力巨大，可以让页面上的所有这种标签都具备一些默认的能力。所以，一般来说用处就是**去掉a标签的下划线、去掉列表项的小圆点**。

```
ul {
    list-style: none;
}

```



### 2、id选择器

* ```
  <p>我是一个段落</p>
  <p id="para2">我是一个段落</p>
  <p>我是一个段落</p>
  <p id="para4">我是一个段落</p>
  <p id="para5">我是一个段落</p>
  
  ```

* 命名规范：

  * ① 不能以数字开头，不能有特殊符号，如果要写符号只能有-和_。

  * ② 同一个名字只能用一次，不能有多个HTML标签是同一个名字。

* CSS中使用井号表示匹配这个id。

  ```
  #para2 {
      color: red;
  }
  
  ```

* 注意`#`后面是一个不能有空格，然后id最好是复制上来的。
* 但是很多公司老大都要求CSS不许使用任何的id选择器。
  * 原因：
    * ① 以后学JavaScript选择元素的时候，最方便的就是id。**所以我们CSS应该将id的命名权交给JS。**
    * ② id选择器，确实太单调，它只能独一无二的选择元素，它不能批量选择元素。

### 3、类选择器

* 下面的6个p标签中，有3个p标签被添加了class="spec"属性，表示他们都是同一类的标签，他们都是spec类的。

* ```
  <p>我是一个段落</p>
  <p class="spec">我是一个段落</p>
  <p class="spec">我是一个段落</p>
  <p>我是一个段落</p>
  <p class="spec">我是一个段落</p>
  <p>我是一个段落</p>
  
  ```

* 类名的命名也是不能有任何符号，只能有-和_。不能以数字开头。

* 在CSS中，使用英语句点符号.给某一个类添加样式。

  ```
  .spec {
      color: red;
  }
  
  ```

* class非常灵活，同一个标签可以同时属于两个类。只需要用空格隔开即可。

  ```
  <p class="spec impor">我是一个段落</p>
  ```

* 现在两个类名spec和import都能够作用在这个p标签上。

  ```
  .spec {
     color: red;
  }
  .impor {
     text-decoration: underline;
  }
  
  ```

* 类选择器是工作中最最常用的选择器。

### 4、交集选择器

*  ul里面携带last这个类的li

* ```
  ul li.last {
      border-bottom: none;
  }
  ```

* 标签的权重怎么数，就是数id的个数、类名的个数、标签名的个数。

## 三、盒子模型

* 老师笔记：https://docs.qq.com/doc/DVUVDQ0xJVHR6VUFu

### 1、什么是盒子模型

* 盒模型：一个盒子，由width、height表示内容区域的宽度和高度，padding表示内边距，border表示边框，margin表示外边距。这样的矩形区域，就是盒模型。
* ![](F:\web\css\images\0.png)
* width属性：表示内容的宽度
* border：表示边框
* padding:表示内容和边框之间的距离
* 快捷键
  - w200 +tab 就是width
  - h200 +tab就是height
  - pdd+tab 就是padding
  - bd +tab就是border
  - mgn +tab就是margin

#### 1、padding详解

* padding表示内边距，边距到文字之间的距离就是padding。

  

  - padding-top  上内边距
  - padding-right 右内边距
  - padding-bottom 下内边距
  - padding-left 左内边距

* 快捷键：

  * pdt 上内边距

  * pdr 右内边距

  * pdb 下内边距

  * pdl 左内边距

* 如果你用空格隔开四个数字

  * `padding: 10px 20px 30px 40px;`

  * 这四个数字表示上、右、下、左。

* 如果你用padding后面写3个数字，比如：

  * `padding: 10px 20px 30px;`

  * 那么上10、右20、下30、左和右相同也是20。

* 如果你用2个数字设置padding，比如

  * `padding:10px 20px;`

  * 那么上10、右20、下和上一样是10、左和右一样是20

#### 2、详解border

* border属性用来设置盒子的边框，盒子可以有边框也可以没有边框。

* ```
  border: 1px solid #000;
  ```

  * 1px 表示边框的宽度，最细就是1px。注意，一定要写px单位，表示像素。
  * solid表示实线，dotted点状线 ，dashed虚线。
  * \#000表示黑色，设计师会给你标注出来颜色值，不用操心。

* 页面上的所有的线，都是边框。
* 小知识点：
  * 单行文本的行高等于盒子高文本垂直居中
  * 多行文本要想垂直居中用padding挤一下，用padding-top和padding-bottom=盒子的高度-自己定义的一行高度*文字行数/2
  * 内容的高度=盒子总高度-padding-border

* 在编写代码的时候，习惯性盒模型属性必须先写，然后写字体相关属性。

### 2、浮动

* 老师笔记：https://docs.qq.com/doc/DVXhQekNyUnpOaGlu

* 浮动是制作并排的div的，天生div是竖直排列的，只要div浮动了，它就能够并排。

* ```
   	<div class="box">

            <div class="c2"></div> → 设置浮动float:right;
            <div class="c3"></div> → 设置浮动float:right;
        </div>
  ```

* 注意，要浮动，所有子盒子就必须都浮动。

* ```
  <style>
          *{
              margin: 0;
              padding: 0;
          }
          .box {
              width: 600px;
              height: 100px;
              border: 1px solid #000;
              margin: 40px auto;
          }
          .box .c1{
              width: 100px;
              height: 100px;
              background-color: orange;
              float:left;
          }
          .box .c2{
              width: 100px;
              height: 100px;
              background-color: purple;
              float:left;
          }
          .box .c3{
              width: 100px;
              height: 100px;
              background-color: skyblue;
              float:left;
          }
      </style>
  
  
  ```

## 四、定位

* 老师笔记：https://docs.qq.com/doc/DVWFQR2JTZVVVVk9y

### 1、相对定位

* 相对于自己原来的位置进行移动

* 注意：

  * **top**： 向**下**移动
  * **bottom**: 向**上**移动
  * **left**： 向**右**移动
  * **right**： 向**左**移动

* ### **注意事项**

  * ① **当前**必须添加position:relative;属性之后，才能使用top、bottom、left、right描述移动。

  * ② 注意left不能和right一起搭配，因为它俩矛盾；top和bottom不能一起搭配，因为它俩矛盾。

  * ③ 有形影分离现象，它本质上真的没有移动，只是渲染效果上把它移动走了。给蓝色盒子施加margin，你会发现，还是能够踹走绿色盒子。它“老家留坑”了。
  * ![](F:\web\css\images\0 (1).png)
  * ④ 相对定位不脱离标准文档流。
    * 什么是脱离标准文档流呢？？就是这个盒子能够彻底地悬空，能够放弃掉它原来的位置。但是相对定位，会在老家“留坑”。所以我们认为相对定位没有标准文档流。

### 2、绝对定位

* 绝对定位就是绝对的坐标位置，通常而言坐标系是网页的角。
* 绝对定位的盒子，书写完绝对定位之后，就能够使用top、left、right、bottom进行一个绝对位置的描述了。
* ![](F:\web\css\images\0 (2).png)

* ### **注意事项**

* ① 绝对定位脱离标准文档流，会完全释放自己的位置，后面的就更够占据它原来的位置。

* ② 既然绝对定位的元素脱离了标准文档流，所以它就不分是行内元素还是块级元素了。脱离标准文档流的元素，一律可以设置宽度和高度。也就是说，行内元素和块级元素是标准文档流中的区分。如果这个元素脱标了，就不再区分了。

* ③ 既然绝对定位的元素脱离了标准文档流，所以它如果不写宽度，就不能自动展开了。

  必须刻意写width: 100%;

### 3、子绝父相

* 一个绝对定位的盒子，不一定总是以网页的角进行定位，可以以祖先盒子进行定位。

* ![](F:\web\css\images\0 (3).png)

* ```
   <div class="box1">→ 没有定位
          <div class="box2"> → 定位了
              <div class="box3"> → 没有定位
                  <div class="box4"> → 没有定位
                      <p></p>  → 绝对定位的盒子
                  </div>
              </div>
          </div>
      </div>
  
  ```

* 离p最近的，它的祖先元素中，box2是第一个定位了的元素，所以p标签以box2进行定位。

* 如果**很多**元素都定位了，那么会找离自己**最近**的：

* ```
     <div class="box1">→ 没有定位
          <div class="box2"> → 相对定位
              <div class="box3"> → 绝对定位
                  <div class="box4"> → 没有定位
                      <p></p>  → 绝对定位的盒子
                  </div>
              </div>
          </div>
      </div>
  ```

* 无论什么定位，只要是祖先元素中，已经定位了的元素，都能够当做绝对定位元素的参考点。比如上例中，box3是离p标签最近的已经定位了的元素。所以参考盒子就是box3。

### 4、绝对定位的应用

* 绝对定位的优点，就是能制作压盖。

* 制作a标签充当按钮，**为了让用户点击的时候不刷新页面**，必须写：

  ```
  <a href="javascript:;"></a>
  ```

  * 或者：

* ```
  <a href="javascript:void();"></a>
  ```

* ### **绝对定位的盒子如何居中（重要，面试常考）**

* ```
  top:50%;//只有这个代码，是这个盒子的顶部处于父盒子的一半
  margin-top=-绝对定位盒子高的一半
  ```

* 要往上“拉”自己身高的一半，此时就要使用margin-top为负数，负的什么？就是负的自己的身高的一半。

* 脱标的方法有三个：浮动、绝对定位、固定定位。
* 相对定位的功能就是微调元素位置和“子绝父相”。子绝父相是给绝对定位的子元素当做定位参考盒子。





* 老师笔记:https://docs.qq.com/doc/DVVdFaWlWSXRFU0Fw

* ## **绝对定位中，top:10px不等价于bottom:-10px**

* 相对定位，top:10px和bottom:-10px是完全等价的。

### 5、定位、浮动的应用

* **绝对定位**擅长做压盖和遮罩，事实上，网页上只要有压盖和遮罩，想都不要想一定是绝对定位来实现。浮动、相对定位都不能实现压盖和遮罩。
* **相对定位**用来微调元素位置，因为老家留坑，所以它也不适合做其他的功能。
* **浮动**用来做布局。因为浮动float:left;咬合的特别紧，页面布局紧凑，容易实现。
